FAVOREATS SPEC SHEET

Bare-Bones App:
1.	Person posts a favor needed (ie. Homework help, fix my iphone screen, mow my lawn, etc) in exchange for food
	       Layout of screen is in a "newsfeed"-esque layout with the newest favor at the top
	       If user wants to post a request, presses a button on the top to submit a request
2.	Other user can “take on” the favor
3.	If original poster of favor accepts, phone numbers are exchanged and a text screen is opened

Stretch goals:
1.	Facebook integration
2.	Filter by everyone/mutual friends/friends
3.	Location filter (1 mile, 3 miles, etc.)
4.	“Counter-offer” ability
